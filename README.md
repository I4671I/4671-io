# 4671's page

一个使用原生 HTML 和 CSS 编写的静态个人博客。

## 新增文章

1. 在 `posts/年份/` 中复制一篇现有文章，并用简短的英文名称命名。
2. 修改页面的标题、描述、日期、阅读时间、摘要和正文。
3. 在 `articles.html` 中加入文章归档条目。
4. 如果希望文章出现在首页，也在 `index.html` 中加入文章卡片。
5. 文章图片放入 `assets/images/posts/文章名称/`，并使用相对路径引用。

全站公共样式位于 `assets/css/main.css`，文章正文样式位于 `assets/css/post.css`。
